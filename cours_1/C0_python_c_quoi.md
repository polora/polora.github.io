# Python c’est quoi ?                                                                                                                         

Python est un langage de programmation créé par Guido Von Rossum dont la première version publique a été publiée en février 1991. Il sert à fabriquer des programmes, c’est à dire un ensemble d’instructions données à un ordinateur pour qu’il exécute certaines actions.

Il existe de très nombreux langages (C, Java, Perl, Pascal, Basic,..).

_A COMPRENDRE :_ 
* Un ordinateur ne comprend que le binaire (une suite de 0 et de 1). 
* On ne peut pas créer un jeu en écrivant que des pages et des pages de 0 et de 1. 
* Il faut donc utiliser un langage de programmation avec des instructions faciles à comprendre (_écrire « Bonjour » sur l’écran_). Ces instructions sont ensuite transformées en binaire.

# Et Pyzo ?           

On peut écrire un programme Python dans n'importe quel éditeur de texte et l'exécuter ensuite en ligne de commande :
```
python3 mon_progamme.py
```
Mais il y a plus pratique : utiliser un IDE (environnement de développement). Dans cet environnement (un logiciel), on peut, sur le même écran, écrire son programme et l’exécuter pour voir ce qu’il fait.
Nous utiliserons __Pyzo__. Il en existe d'autres comme _PyCharm_, _Wing IDE_ par exemple.
