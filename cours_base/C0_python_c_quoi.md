# Python c’est quoi ?                                                                                                                         

Python est un langage de programmation. Il sert à fabriquer des programmes, c’est à dire un ensemble d’instructions données à un ordinateur pour qu’il exécute certaines actions.

Il existe de très nombreux langages (C, Java, Perl, Pascal, Basic,..).

_A COMPRENDRE :_ 
* Un ordinateur ne comprend que le binaire (une suite de 0 et de 1). 
* On ne peut pas créer un jeu en écrivant que des pages et des pages de 0 et de 1. 
* Il faut donc utiliser un langage de programmation avec des instructions faciles à comprendre (_écrire « Bonjour » sur l’écran_). Ces instructions sont ensuite transformées en binaire.

# Et Pyzo ?                                                                                                                               

Pyzo un environnement de développement (IDE). Dans cet environnement (un logiciel), on peut, sur le même écran, écrire son programme et l’exécuter pour voir ce qu’il fait.
